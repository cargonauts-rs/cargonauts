    /// preflight request can be cached in a preflight result cache.
    ///
    /// # ABNF
    /// ```plain
    /// Access-Control-Max-Age = \"Access-Control-Max-Age\" \":\" delta-seconds

    /// # Example values

    ///
    /// # Examples
    /// ```
